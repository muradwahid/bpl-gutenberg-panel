# Custom Components
The ``bpl-gutenberg-panel`` package provides a collection of reusable React components for building custom Gutenberg blocks in WordPress. These components include various UI elements such as background, overlay, gradient pickers, and more, designed to facilitate the development of rich and interactive block interfaces within the WordPress block editor.
